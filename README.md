# MyZip Tool - Smart File Compressor

A web-based file compression utility that allows users to zip multiple files efficiently. Built using JavaScript, HTML5, and CSS3.

## Features
- Multi-file selection and zipping.
- Lightweight and fast processing.
- Clean and intuitive User Interface.

## Technologies Used
- **Frontend:** HTML5, CSS3
- **Logic:** JavaScript (using JSZip library)

## How to Run
1. Clone this repository.
2. Open `index.html` in any web browser.
3. Select files and click "Compress" to download the zip.
